Code 208 - Missing PN Tag or Sign

Code 208 – Reference Image
Primary neutral conductor requiring PN identification sign
Standard Requirement
When to Apply a PN Sign

When the primary phase conductor and the primary neutral conductor cannot be visually distinguished from each other, SDG&E electric standards require that the primary neutral conductor be identified by a PN sign.

Click to expand
Code Usage
When to Use Code 208

Use Code 208 — "Missing PN Tag or Sign" — whenever the required PN identification sign is absent from the primary neutral conductor on a structure where the phase and neutral conductors are not easily distinguishable.

Maintenance Action
No On-Site Maintenance Required

This condition does not require on-site maintenance. It should be documented and flagged for correction during the next scheduled maintenance visit or planned work at the structure.
